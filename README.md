## 📌 Overview

Campus Connect is a full-stack community platform designed for students to:

* 🧠 Post problems
* 💡 Share solutions
* 🤝 Collaborate with peers
* 🏆 Gain recognition through contributions

This project is a rebranded and enhanced version of a problem-solving platform, built using modern web technologies.

---

## 🛠️ Tech Stack

* **Frontend:** EJS, Bootstrap 5, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)
* **Authentication:** Passport.js
* **File Uploads:** Multer + Cloudinary
* **Session Storage:** MongoStore

---

## ✨ Features

* 🔐 User Authentication (Login/Signup)
* 📢 Post & Browse Problems
* 💬 Add Solutions
* 👍 Upvote Solutions
* 🏆 Leaderboard System
* 👤 User Profiles (Skills, GitHub, LinkedIn)
* 🌙 Modern UI with Glassmorphism Design
* 📱 Fully Responsive Design

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/PrinceYadav001/Campus_Connect.git
cd Campus_Connect
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file and add:

```
MONGO_URI=your_mongodb_connection
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
SESSION_SECRET=your_secret
```

---

### 4️⃣ Run the project

```bash
node app.js
```

App will run at:

```
http://localhost:3000
```
